# Student Management - Frontend

This is the Streamlit-based news classification dashboard.

## 🚀 How to Activate

### 1. Initialize/Sync Environment
Ensure you are in the `frontend` directory and sync dependencies:
```bash
uv sync
```

### 2. Activate Virtual Environment
```bash
source .venv/bin/activate
```

### 3. Run the Dashboard
```bash
uv run streamlit run app.py
```

---
**Note:** The dashboard expects the model to be located at `../backend/news_classifier_model.joblib`.
